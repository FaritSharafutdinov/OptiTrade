## Инструкция

Последовательно запусти два скрипта: data_fetch -> feature_engineering.

Первый скрипт соберет данные OHLCV + OI

Второй скрипт обогатит датасет техническими индикаторами и time-based фичами

Изменения датасета в результате FE: 

1)убраны OHLCV ввиду нестационарности данных а также корреляции между этими фичами.

2)добавлены фичи log return ввиду стационарности таких данных при высоком MI score с Close price

3)добавлен индекс sp500 log return как фича

4)time based features получили синус-косинусную форму

5)устранил data leakage индикаторов

6)добавил некоторые тех индикаторы и убрал некоторые старые
